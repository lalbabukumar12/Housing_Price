📊 Visualizing Housing Market Trends using Tableau
📌 Project Title

Visualizing Housing Market Trends: An Analysis of Sale Prices and Features using Tableau

📖 Project Overview

This project analyzes housing market trends using Tableau data visualization techniques. The goal of the project is to understand how different housing features influence house sale prices.

By analyzing housing data, this project identifies patterns and relationships between sale price and important property features such as house age, renovation status, number of bedrooms, bathrooms, floors, and basement area.

The interactive Tableau dashboard helps users explore housing market trends and gain meaningful insights from the data.

🎯 Problem Statement

The objective of this project is to analyze housing market trends and understand how different property features affect house sale prices.

Using Tableau visualizations, the project focuses on uncovering patterns related to:

House Age

Renovation Status

Number of Bedrooms

Number of Bathrooms

Number of Floors

Basement Area

The project aims to provide clear visual insights into the factors that influence housing prices.

📂 Dataset Information

The dataset used in this project was obtained from Kaggle, a popular platform for open datasets.

Dataset Features

The dataset contains several housing-related attributes including:

Sale Price

Basement Area (Sqft)

Years Since Renovation

Number of Bedrooms

Number of Bathrooms

Number of Floors

House Age

Data Format

File Type: CSV

Source: Kaggle Housing Dataset

🛠 Data Preprocessing

Before creating visualizations, the dataset was cleaned and prepared to ensure accurate analysis.

Steps Performed

Checked for missing values

Verified correct data types

Renamed fields for better readability

Created calculated fields such as:

House Age

Years Since Renovation

Validated dataset consistency

These preprocessing steps ensured that the data was clean and ready for visualization.

🔍 Key Insights

From the analysis, several important insights were discovered:

Houses with larger basement areas tend to have higher sale prices.

Renovated houses often have higher market value.

Houses with more bedrooms and bathrooms generally sell at higher prices.

House age influences price distribution in the housing market.
